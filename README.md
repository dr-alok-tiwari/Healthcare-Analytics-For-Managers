# Healthcare Analytics for Managers

A complete Streamlit learning and career-readiness app for PGDM-HCM / MBA / Executive learners.

## What it includes

- Healthcare analytics mindset
- Executive KPI dictionary
- **Manager Decision Room 1200+** with guaranteed filter coverage
- 200 detailed business cases with 20 cases per category
- Analytics Sandbox with 50 datasets
- Visual analytics studio
- 500 MCQs with click-to-reveal answers
- 500 interview questions with filters
- Custom career roadmap generator
- Faculty notes
- Detailed About the Developer section with portfolio link

## How to run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Notes

All datasets are simulated and built into the app. The app is designed for classroom teaching and does not require paid APIs or external files.
